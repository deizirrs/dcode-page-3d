# 🌐 3D Landing Page

Um projeto moderno e interativo de página web 3D, criado com HTML, CSS e animações 3D usando **Spline** — sem codificação complexa. Ideal para quem deseja aprender ou implementar elementos 3D responsivos e interativos em seus projetos web.

## 🚀 O que este projeto entrega:

✅ Integração fluida de modelos 3D com **Spline**  
✅ Layout limpo e responsivo com **HTML e CSS**  
✅ Animações suaves e efeitos interativos  
✅ Design responsivo e adaptável para qualquer dispositivo  

## 🎨 Tecnologias utilizadas:

- **HTML5**
- **CSS3**
- **Spline 3D Viewer**

## 📌 Funcionalidades implementadas:

- Integração de animação 3D via `<spline-viewer>`
- Estilização personalizada da seção de conteúdo
- Animações de recarga e efeitos visuais
- Responsividade total para dispositivos móveis e tablets
- Melhorias de layout para experiência otimizada
